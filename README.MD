# Mario (MonoGame)

A simple Mario-style platformer built from scratch using C# and MonoGame.

## Features
* Custom 2D physics system (movement, gravity, collisions)
* Tile-based level system
* Basic AI behaviors for enemies
* Animation system (sprite sheets)
* Sound effects and background music
* Scene system (menu, gameplay, win/lose screens)

## Architecture
* Separation between game logic and rendering
* Entity-based design
* Simple collision rules system
* Factory-based entity creation

## Purpose
This project was built as a learning exercise to explore:

* game loop design
* physics and collision handling
* tile-based world representation
* basic game architecture and separation of concerns

# Notes
* This is a learning project
* Some systems are simplified
* Focus was on architecture rather than content
